Introduction To Azure

  • Type: Development
  • Partner: IEEE
  • Date: 5 March, 2021

Summary

An introductory hands-on workshop on the basics of Databases and Azure Database services. As one jumps into the world of Data Science & Machine Learning with all the given hype, the very first thing they go through is managing and retrieving data in an efficient way. At initial level Database Systems comes in handy. This workshop offers an easier way for beginners to learn basics of Database management systems, perform CRUD on Databases with demonstration of Cloud based Database creation.
